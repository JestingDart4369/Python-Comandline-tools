# Security Policy

## Supported Versions

Only the **latest version** of this repository is supported with security updates.

| Version | Supported |
| ------ | --------- |
| Latest | ✅ Yes    |
| Older | ❌ No     |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report the vulnerability responsibly by contacting the repository maintainer directly.

### How to Report
Please include:
- A clear description of the vulnerability
- Steps to reproduce the issue (if applicable)
- Affected files or modules
- Potential impact of the issue
- (Optional) Suggestions for mitigation or fixes

### Where to Report
- Via **GitHub private message**, or  
- Via **email** (if listed on the maintainer’s GitHub profile)

---

## Security Considerations

This repository contains tools that may interact with:
- External APIs
- Login credentials
- Downloaded or generated files

Please note:
- **API keys, passwords, and tokens must never be committed**
- Sensitive files should be ignored using `.gitignore`
- Example or placeholder data should be used for demonstrations

---

## Disclaimer

This project is intended for **educational and personal use only**.  
It is **not designed for production environments** and should not be used to handle sensitive data without additional security measures.

---

Thank you for helping keep this project secure.
